 Product Landing Page - Original Trombones

A responsive product landing page for a fictional trombone company, built as part of a freeCodeCamp project.

 📋 Overview

This landing page showcases handcrafted trombones with a clean, professional design. It features a fixed navigation header, product information sections, an embedded video, and pricing cards for different trombone models.

 🚀 Features

- **Responsive Navigation Bar**: Fixed header with smooth scrolling links
- **Email Capture Form**: Newsletter signup with email validation
- **Feature Grid**: Three key selling points with iconography
- **Product Showcase**: Embedded YouTube video demonstration
- **Pricing Cards**: Three product tiers with specifications
- **Footer**: Privacy policy, terms, and contact links

 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

 📁 File Structure

```
project/
│
├── index.html           Main HTML document
└── styles.css           Stylesheet
```

 🎨 Design Elements

 Color Scheme
- Background: `#eee` (light gray)
- Accent: Dark Orange (for icons)
- Headers: `#ddd` (light gray)
- Text: Black and `#444` (dark gray)

 Typography
- Google Fonts: Anton, Baskervville, Raleway
- Responsive font sizing

 Icons
- Font Awesome 5.8.2
- Fire, Truck, and Battery icons for features

 📱 Responsive Design

The page includes:
- Flexible grid layouts
- Responsive images
- Mobile-friendly navigation
- Media query breakpoints

 🔧 Installation & Usage

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process required - it's ready to use!

 📝 Page Sections

1. **Header**: Company logo and navigation
2. **Hero Section**: Call-to-action with email signup
3. **Features**: Premium materials, fast shipping, quality assurance
4. **How It Works**: Product demonstration video
5. **Pricing**: Three trombone models with pricing
6. **Footer**: Legal links and copyright
